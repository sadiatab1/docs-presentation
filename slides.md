# ⚡ Common Issues  

**Images Not Loading**  
- Use relative paths  
- Add `--allow-local-files` for local export  

**Font Problems**  
- Embed web fonts in custom theme  

**Build Errors**  
- Check Node.js version  
- Install dependencies  
- Use `--verbose`
